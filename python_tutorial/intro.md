## python作为一个计算器

```python
>>> (50 - 5*6) / 4
5.0

>>> 17 / 3  # 实数除法
5.666666666666667
>>>
>>> 17 // 3  # 整数除法
5
>>> 17 % 3  # the % 取模
2
>>> 2 ** 7  #  指数
128
```
用等于符号 (=) 可以给一个值起一个名字，叫做变量
```python
>>> pi = 3.1415926
>>> r= 2.5
>>> area = pi*r*r/2
>>> r = "hello"  #变量可以表示不同类型的量，现在r表示了一个字符串。变量类型根据它指向的值的类型而变化，称之为“动态类型"。
                 # 因此，python称为动态语言
```
